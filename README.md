# File Finder

A fast and simple Windows file search utility built with **Python** and **PyQt6**.

File Finder helps you locate files on your computer by searching their names. If multiple files share the same name, you can easily choose the correct one and perform actions such as opening the file, revealing its location, copying its path, or deleting it.

---

## Features

*  Search for files by name
*  Displays multiple matching locations
*  Open the selected file
*  Open the file's containing folder
*  Copy the full file path
*  Delete files with confirmation
*  Modern PyQt6 graphical interface
*  Custom title bar
